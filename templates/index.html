Tentang Sistem

Sistem ini dibuat sebagai alat uji dari skripsi yang berjudul Klasifikasi Interferensi Jaringan Menggunakan Light Gradient Boosting Machine (LightGBM) Dengan Recursive Featrue Elimination (RFE) Berbasis Logistic Regression (LR)

Untuk mempermudah pengguna, silahkan untuk membaca petunjuk penggunaan sistem dibawah ini

  • 01 Pilih Menu Klasifikasi

    Menu ini berguna untuk masuk kehalaman klasifikasi. Anda juga dapat klik "Mulai Klasifikasi" pada halaman dashboard.

  • 02 Masukan Dataset Testing

    Pada halaman ini anda diminta untuk mengupload dataset testing dengan jenis csv.

  • 03 Klik Submit

    Tunggu sejenak sementara sistem melakukan analisis berdasarkan dataset yang sudah di upload. Selanjutnya hasil prediksi akan muncul di layar.
